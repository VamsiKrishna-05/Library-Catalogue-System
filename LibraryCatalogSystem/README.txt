# Java Console-Based Library Catalog System

## Features
- Add, search, list, update, and delete books
- Save/load data from a file (`books.txt`)
- Prevent duplicate ISBNs
- Sort books by title or author
- Fast lookup using HashMap
- Menu-driven user interface

## How to Run

1. Open terminal in the folder
2. Compile:
   javac Book.java Library.java LibraryApp.java
3. Run:
   java LibraryApp

books.txt will be created in the same folder to store book data.

Enjoy!
